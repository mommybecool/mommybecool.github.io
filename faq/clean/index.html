Πως μπορώ να καθαρίσω την πλαστελίνη από τα αντικείμενα;

Καθώς τα παιδιά παίζουν η πλαστελίνη μπορεί να ξεραθεί είτε πάνω στα εργαλεία, είτε πάνω στα διάφορα μέρη των kit παιχνιδιού. Πλένοντας τα με νερό, θα γίνουν σαν καινούρια! Στο μόνο που χρειάζεται να δώσετε προσοχή είναι τα ξύλινα μέρη που είναι βαμμένα, καθώς χρησιμοποιούμε μη τοξικά χρώματα, αλλά δεν είναι λουστραρισμένα, γιατί έρχονται σε επαφή με παιδιά, σε αυτά συνιστούμε να τα καθαρίσετε με πανάκι και νερό χωρίς να τα τρίψετε δυνατά καθώς υπάρχει περίπτωση αποχρωματισμού. Φροντίστε για όλα τα αντικείμενα να στεγνώσουν καλά πριν έρθουν εκ νέου σε επαφή με την πλαστελίνη.
